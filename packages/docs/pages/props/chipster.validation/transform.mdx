# Transform

The `transform` prop allows you to modify the input value before it's added as a chip.

## Type
typescript
`(value: string) => string`

## Usage
```tsx filename="ChipsterExample.tsx" {11} copy
import { Chipster } from '@micoblanc/chipster';

function ChipsterExample() {
  const transformInput = (value: string) => {
    // Convert to uppercase and trim whitespace
    return value.toUpperCase().trim();
  };

  return (
    <Chipster
      transform={transformInput}
      // other props...
    />
  );
}
```

## Notes

- The `transform` function is applied to the input value before validation and addition to the chip list.
- Use this prop to standardize input, remove unwanted characters, or apply any other text transformations.
- The transformed value is what gets added as a chip, not the original input.

It's used when adding new items to ensure consistent formatting of chip text.
