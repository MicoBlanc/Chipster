# Show Error Message

The `showErrorMessage` prop determines whether error messages from validation rules should be displayed.

## Type
typescript
`boolean`

## Usage
```tsx filename="ChipsterExample.tsx" {6} copy
import { Chipster } from '@micoblanc/chipster';

function ChipsterExample() {
  return (
    <Chipster
      showErrorMessage={true}
      // other props...
    />
  );
}
```
## Notes

- When set to `true`, error messages from failed validation rules will be displayed below the input field.
- When set to `false`, validation still occurs, but error messages are not shown to the user.
- This prop is useful for controlling the visibility of error feedback in different contexts.
- The default value is `true`.
