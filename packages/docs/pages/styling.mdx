# Styling Guide for Chipster

Chipster is designed to be highly customizable through className props and theme support. This guide explains the available styling options.

## Component-Based Styling

Each Chipster component accepts className props for custom styling:

```tsx
<Chipster className="container-class">
  <Chipster.ItemList 
    className="items-class"
    itemClassName="item-class"
    removeButtonClassName="remove-btn-class"
  />
  <Chipster.Input 
    className="input-class"
  />
  <Chipster.Validation
    errorClassName="error-class"
  />
  <Chipster.Suggestions 
    className="suggestions-class"
  />
</Chipster>
```

## Theme Support

Chipster supports light and dark themes out of the box:

```tsx
<Chipster theme="light"> // Default
  {/* ... components */}
</Chipster>

<Chipster theme="dark">
  {/* ... components */}
</Chipster>
```

## Styling with Tailwind

Here's an example using Tailwind CSS classes:

```tsx
<Chipster className="bg-white dark:bg-gray-800 p-4 rounded-lg shadow-sm">
  <Chipster.ItemList 
    className="flex flex-wrap gap-2"
    itemClassName="bg-blue-100 dark:bg-blue-900 text-blue-900 dark:text-blue-100 px-3 py-1 rounded-full"
    removeButtonClassName="ml-2 text-blue-500 hover:text-blue-700"
  />
  <Chipster.Input 
    className="w-full mt-2 px-3 py-2 bg-transparent border-none focus:outline-none"
    placeholder="Type and press Enter"
  />
  <Chipster.Validation
    errorClassName="text-red-500 text-sm mt-1"
  />
  <Chipster.Suggestions 
    className="mt-1 bg-white dark:bg-gray-800 shadow-lg rounded-lg overflow-hidden"
  />
</Chipster>
```

## Available className Props

### Chipster
- `className`: Applied to the main container

### Chipster.ItemList
- `className`: Applied to the items container
- `itemClassName`: Applied to individual items
- `removeButtonClassName`: Applied to remove buttons
- `removeIcon`: Custom remove button icon
- `iconClassName`: Applied to item icons

### Chipster.Input
- `className`: Applied to the input element

### Chipster.Validation
- `errorClassName`: Applied to error messages

### Chipster.Suggestions
- `className`: Applied to suggestions container

## CSS Modules

Chipster uses CSS Modules internally. The main classes are:

```css
.container {
  /* Main container styles */
}

.input {
  /* Input field styles */
}

.item {
  /* Individual item styles */
}

.itemRemove {
  /* Remove button styles */
}

.suggestions {
  /* Suggestions container styles */
}

.error {
  /* Error message styles */
}

/* Dark theme variants */
.containerDark {
  /* Dark theme container styles */
}

.inputDark {
  /* Dark theme input styles */
}

.itemDark {
  /* Dark theme item styles */
}
```

## Best Practices

1. Use the composable nature of Chipster to style each component independently
2. Leverage Tailwind classes for quick styling
3. Use the theme prop for dark mode support
4. Apply custom classes for specific styling needs
5. Keep styles consistent across all Chipster components

## Example: Complete Styled Component

```tsx
export default function StyledChipster() {
  return (
    <Chipster 
      className="max-w-xl mx-auto bg-white dark:bg-gray-800 p-4 rounded-lg shadow-sm"
      theme="light"
    >
      <Chipster.ItemList 
        className="flex flex-wrap gap-2"
        itemClassName="flex items-center bg-blue-100 dark:bg-blue-900 text-blue-900 
                     dark:text-blue-100 px-3 py-1 rounded-full"
        removeButtonClassName="ml-2 text-blue-500 hover:text-blue-700"
      />
      <Chipster.Input 
        className="w-full mt-2 px-3 py-2 bg-transparent border-none 
                  focus:outline-none placeholder-gray-400"
        placeholder="Type and press Enter"
      />
      <Chipster.Validation
        errorClassName="text-red-500 text-sm mt-1 px-2"
      />
      <Chipster.Suggestions 
        className="mt-1 bg-white dark:bg-gray-800 shadow-lg rounded-lg 
                  overflow-hidden border border-gray-200 dark:border-gray-700"
      />
    </Chipster>
  )
}
```

## Styling Individual Items

You can style individual items using the `itemClassName` prop:

```tsx
<Chipster.ItemList 
  itemClassName="
    flex items-center gap-2 
    bg-blue-100 hover:bg-blue-200 
    px-3 py-1 rounded-full 
    transition-colors
  "
  removeButtonClassName="
    p-1 rounded-full 
    hover:bg-blue-200 
    transition-colors
  "
/>
```
