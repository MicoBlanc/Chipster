# Chipster

Chipster is a flexible and powerful multi-entry input component for React. It's perfect for managing tags, email addresses, or any list-based input scenario where you need to handle multiple entries efficiently.

## Basic Example

<ChipsterExample />

## Features

- Customizable input and chip styles
- Validation rules for input
- Autocomplete suggestions
- Animations for chip removal
- Accessibility support
- TypeScript support
- Flexible API with `useChipster` hook

## Installation

Install Chipster via npm:

```bash filename="Install library" {3} copy
npm install @micoblanc/chipster
```
## How to Use

To use Chipster in your React application, import it and add it to your JSX. Here's a basic example:
```js filename="Chipster.jsx" {3} copy
import { Chipster } from '@micoblanc/chipster';

function ChipsterComponent() {

return (
    <Chipster
        placeholder="Enter emails"
        onAdd={(item) => console.log('Added:', item)}
        onRemove={(item) => console.log('Removed:', item)}
        getIcon={(value) => value.includes('@') ? '✉️' : null}
        maxItems={5}
        allowDuplicates={false}
    />
    );
}
```

## API Reference

| Prop | Type | Description |
| ---- | ---- | ----------- |
| `placeholder` | `string \| React.ReactNode` | Text or element to display when the input is empty |
| `onAdd` | `(item: string) => void` | Callback function called when an item is added |
| `onRemove` | `(id: string) => void` | Callback function called when an item is removed |
| `validationRules` | `Array<{ test: (value: string) => boolean, message: string }>` | Array of rules to validate input |
| `getIcon` | `(value: string) => React.ReactNode` | Function to get an icon for each chip based on its value |
| `maxItems` | `number` | Maximum number of items allowed |
| `allowDuplicates` | `boolean` | Allow or disallow duplicate items |
| `className` | `string` | CSS class for the main container |
| `inputClassName` | `string` | CSS class for the input element |
| `chipClassName` | `string` | CSS class for each chip |
| `errorClassName` | `string` | CSS class for error messages |
| `getSuggestions` | `(input: string) => string[]` | Function to get autocomplete suggestions based on input |
| `transform` | `(value: string) => string` | Function to transform input before adding as a chip |
| `caseSensitive` | `boolean` | Determine if chip values are case-sensitive |
| `showErrorMessage` | `boolean` | Show or hide error messages |
| `exitAnimation` | `"fade" \| "slideLeft" \| "slideRight" \| "fadeSlideLeft" \| "fadeSlideRight"` | Animation style for removing chips |

Customize Chipster's behavior and appearance by adjusting these props to fit your specific use case.
